# swift-cli-sieve-atkins

## Description
Find all primes less than N.

## Tech stack
- bash
- swift 5.3

## Docker stack
- docker-cli
- swift:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
